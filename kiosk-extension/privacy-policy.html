zmNinjaNG Kiosk Config - Privacy Policy

Last updated: March 29, 2026

What this extension does

This extension reads configuration set by your organization's administrator via Google Admin Console and delivers it to the zmNinjaNG web app running on designs-for-learning.github.io. This enables zero-touch setup of Chromebook kiosk devices.

Data collected

This extension does not collect, transmit, or store any personal data. It reads enterprise managed configuration (chrome.storage.managed) which is set by your organization's IT administrator, not by end users.

Data handling

  • Configuration data (server URL, credentials, kiosk settings) is read from chrome.storage.managed and written to the web app's localStorage on the local device only.
  • No data is sent to any external server, analytics service, or third party.
  • No browsing history, cookies, or user activity is accessed or recorded.

Third-party services

This extension does not use any third-party services, analytics, or tracking.

Contact

For questions about this privacy policy, contact the repository maintainers at github.com/Designs-for-Learning/zmNg.
